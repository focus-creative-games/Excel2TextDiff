# Excel2TextDiff

一个方便diff excel族(xls,xlsx,csv)文件的工具。将excel文件转为文本文件，然后再调用diff程序，极其方便直观地对比excel文件的变化。非常有用！

**===需要安装 [.net 8 sdk](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)或更高版本===**
